## 功能包概述

### face_rec 功能包

- 基于 dlib 深度学习模型，准确率 99.38%
- 提供 **Service** 和 **Topic** 两种调用方式
- 依赖：`pip install dlib==19.19.0`、`pip install face_recognition`

### 自定义消息

**face_rec/msg/face_data.msg**：
```
string name         # 姓名
float64 xmin        # 人脸框像素坐标
float64 xmax
float64 ymin
float64 ymax
```

**face_rec/msg/face_results.msg**：
```
uint8 num                        # 识别到的人脸数量
face_rec/face_data[] face_data   # 人脸数据
```

### 自定义服务

**face_rec/srv/recognition_results.srv**：
```
int8 mode           # 1=识别相机画面, 2=识别图片
string image_path   # mode=2 时的图片路径
---
face_rec/face_results result  # 识别结果
bool success                  # 调用成功
```

## 使用方式

### 方式一：Service 调用

```bash
# 启动服务
roslaunch face_rec face_rec_service.launch

# 调用识别（mode=1 从相机, mode=2 从图片）
rosservice call /face_recognition_results "mode: 1
image_path: ''"
```

### 方式二：Topic 订阅

```bash
# 启动节点
roslaunch face_rec face_rec_topic.launch

# 查看结果（帧率约 1.4fps）
rqt_image  # 选择 /camera/face_recognition 话题
```

## 编程调用

```cpp
#include <ros/ros.h>
#include <face_rec/recognition_results.h>

int main(int argc, char **argv)
{
    ros::init(argc, argv, "face_rec_ex");
    ros::NodeHandle n;

    ros::ServiceClient face_rec_client =
        n.serviceClient<face_rec::recognition_results>("face_recognition_results");

    face_rec::recognition_results face_rec_data;
    ros::service::waitForService("face_recognition_results");

    while (ros::ok())
    {
        face_rec_data.request.mode = 1;  // 从相机识别
        face_rec_client.call(face_rec_data);

        if (face_rec_data.response.success)
        {
            int face_num = face_rec_data.response.result.num;
            std::cout << "我看到了" << face_num << "个人脸, 他们分别是：" << std::endl;
            for (int i = 0; i < face_num; i++)
            {
                std::string name = face_rec_data.response.result.face_data[i].name;
                std::cout << name << " ";
            }
            std::cout << std::endl;
        }
    }
    return 0;
}
```

## 添加人脸库

1. 使用"茄子"拍照软件拍摄照片
2. 在 `face_data/` 目录下创建以人名命名的文件夹
3. 将照片放入对应文件夹
4. 重启 face_rec 服务即可识别

> ⚠️ 启动 face_rec 服务前需关闭"茄子"拍照软件，否则相机调用失败
