## Gazebo 仿真平台

- 开源机器人仿真平台，由 OSRF（Open Source Robotics Foundation）维护
- 与 ROS 兼容性最好
- 支持 ODE 物理引擎
- 可模拟传感器：激光雷达、摄像头、IMU 等

### 启动仿真机器人

```bash
roslaunch bobac3_description gazebo.launch
```

> 会同时打开 Gazebo（仿真环境）和 Rviz（数据可视化）

> ⚠️ 虚拟机中 Gazebo 容易崩溃，如遇问题执行：
> ```bash
> echo "export SVGA_VGPU10=0" >> ~/.bashrc
> ```

### Gazebo 基本操作

| 操作 | 功能 |
|------|------|
| 按住左键移动鼠标 | 移动场景 |
| 按住滚轮移动鼠标 | 旋转场景 |
| 拨动滚轮 | 放大/缩小 |

## 仿真机器人话题

启动后可用 `rostopic list` 查看，主要话题：

| 话题 | 内容 |
|------|------|
| `/scan` | 激光雷达数据 |
| `/bottom_camera/*` | 底部相机数据 |
| `/imu` | IMU 数据 |
| `/berxel_base/*` | 深度相机数据 |

### 查看激光雷达数据

```bash
rostopic echo /scan
```

---

## Rviz 数据可视化

1. 打开 Rviz → 点击左下角 **Add**
2. 选择 **By topic**
3. 添加 `LaserScan` → 可视化激光雷达数据

Rviz 可以直观展示各种传感器数据，是 ROS 开发中最重要的调试工具之一。
