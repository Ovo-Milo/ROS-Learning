### 2. 导入虚拟机系统

1. 将下载的 `ubuntu元宝` 文件夹解压（磁盘至少需要 40G 以上空间）
2. 打开 VMware → 点击 **打开虚拟机**
3. 选择 `ubuntu元宝/Ubuntu 64 位.vmx` 文件 → 点击打开
4. 点击 **开启此虚拟机**

> 💡 遇到 "我已复制该虚拟机" 提示时，选择"我已复制该虚拟机"即可
